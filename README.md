# Tesina
En este repositorio se encuentran los scripts utilizados durante la realización de mi tesina de Licenciatura en Física
En la carpeta "Potencial" se encuentran los códigos utilizados para realizar las gráficas presentadas en la tesina. 
En la carpeta "RIFTS" se encuentran los códigos que fueron necesarios para la implementación de la técnica RIFTS
En la carpeta "Processado de Imágenes" se encuentran los codigos utilizados para el análisis de las imágenes del mapeo superficial presentado en la tesina.
